🎯 Descripción general del aprendizaje

Este proyecto tiene como objetivo que el alumnado desarrolle competencias digitales básicas para desenvolverse de forma autónoma, segura y crítica en su vida cotidiana.

A través de actividades prácticas y significativas, el alumnado aprenderá a:

Utilizar el correo electrónico (enviar, recibir y adjuntar archivos).
Comunicarse mediante herramientas digitales.
Buscar información en internet de forma guiada.
Identificar problemas técnicos básicos.
Aplicar estrategias para resolver incidencias de manera autónoma o colaborativa.
Usar la tecnología de forma segura (protección de datos y prevención de fraudes).
Se emplean metodologías activas, aprendizaje significativo y acompañamiento emocional, fomentando la confianza y la autonomía progresiva.
🧠 Justificación pedagógica

Este proyecto surge a partir de la reflexión realizada mediante el ciclo de Gibbs sobre mi práctica docente en el aula de personas adultas.

Se detectaron dificultades clave:

Baja competencia digital
Inseguridad ante el uso de la tecnología
Dependencia del docente
Problemas técnicos básicos frecuentes
El análisis mostró que estas dificultades son tanto técnicas como emocionales, lo que hace necesario:

No dar por supuestos conocimientos previos
Enseñar estrategias de resolución de problemas
Incorporar apoyos visuales y secuenciación
Fomentar la autonomía del alumnado
Como respuesta, este proyecto incluye:

✔ Actividades guiadas paso a paso
✔ Estrategias para resolver problemas técnicos
✔ Aprendizaje entre iguales
✔ Materiales adaptados y accesibles
✔ Refuerzo emocional para reducir el miedo al error

Además, se atienden aspectos éticos como:

Reducción de la brecha digital
Inclusión social
Uso seguro de la tecnología
Se asume la incertidumbre del uso tecnológico mediante una planificación flexible y adaptativa.
🏁 Producto final del alumnado

El alumnado será capaz de:

Enviar un correo electrónico completo de forma autónoma, incluyendo:

Redacción del mensaje
Inclusión de destinatario
Archivo adjunto
Resolución básica de problemas durante el proceso
✨ Enfoque metodológico

Aprendizaje práctico
Ritmo adaptado
Atención individualizada
Refuerzo positivo
Aprendizaje significativo y funcional
